合同细则

1.1 甲方完成作业后，有权要求乙方现场支付剩余服务费；若乙方迟延支付植保服务费的，甲方有权终止服务，并同时有权按逾期每日千分之三的标准支付违约金直至清偿之日。

第二条乙方的权利及义务

2.1 乙方须按照合同的约定支付植保服务费。

2.2 乙方有权对甲方提供的植保服务过程进行监督。

2.3 乙方积极配合甲方作业，提供转运物资的便利条件，维护现场秩序，让围观人员保持与飞机10米以上的安全距离。

2.4 若乙方对植保服务有异议的，应在植保服务全部完成后三天内及时通知甲方联系人，如实反馈问题。

第三条责任免除

以下原因造成的损失，甲方不承担责任。

3.1 由乙方或第三方提供的农药，造成的施药效果不佳而造成作物的损失

3.2 不可抗拒的自然灾害（如洪水、冰雹等）造成的作物损失。

3.3 因乙方没有及时对作业现场的无关人员进行清场，或者对作业环境没有及时清理造成的人员伤害或其他损失。
